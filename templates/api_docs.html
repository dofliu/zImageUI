認證方式

所有 /api/v1/ 端點需要 API 金鑰認證。您可以在儀表板建立金鑰。

HTTP Header 
X-API-Key: zimg_your_api_key_here

或使用 Query Parameter：

Query Parameter
GET /api/v1/history?api_key=zimg_your_api_key_here

快速上手

使用 cURL 生成第一張圖片：

cURL
curl -X POST http://localhost:5000/api/v1/generate \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{
    "prompt": "a beautiful sunset over mountains, digital art",
    "width": 768,
    "height": 768
  }'
Python
import requests

response = requests.post(
    "http://localhost:5000/api/v1/generate",
    headers={"X-API-Key": "YOUR_API_KEY"},
    json={
        "prompt": "a beautiful sunset over mountains",
        "width": 768,
        "height": 768,
        "output_format": "url"
    }
)

data = response.json()
print(f"Image: {data['image_url']}")
print(f"Seed: {data['seed']}")
JavaScript
const response = await fetch('/api/v1/generate', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    prompt: 'a beautiful sunset over mountains',
    width: 768,
    height: 768
  })
});

const data = await response.json();
console.log(data.filename, data.seed);
POST /api/v1/generate

生成一張 AI 圖片。

Request Body

參數類型必填說明
promptstring是圖片描述提示詞
negative_promptstring否負面提示詞
widthinteger否圖片寬度 (預設 768)
heightinteger否圖片高度 (預設 768)
seedinteger否隨機種子（重現結果）
modelstring否指定模型 ID
output_formatstring否base64 (預設) 或 url

Response

200 OK
{
  "success": true,
  "filename": "api_20260320_143022.png",
  "prompt": "a beautiful sunset",
  "seed": 1234567890,
  "width": 768,
  "height": 768,
  "model": "z-image-turbo",
  "image": "data:image/png;base64,..."
}
GET /api/v1/history

取得歷史生成記錄。

Query Parameters

參數類型說明
limitinteger回傳數量 (預設 50)
offsetinteger分頁偏移量 (預設 0)
GET /api/v1/models

列出所有可用的 AI 模型。

POST /api/queue/submit

提交生成任務到非同步佇列。適合批量生成或長時間運算。

Request Body

{
  "type": "generate",
  "priority": 0,
  "params": {
    "prompt": "a cat in space",
    "width": 1024,
    "height": 1024,
    "project_id": "abc12345"
  }
}

Response

{
  "success": true,
  "task": {
    "id": "a1b2c3d4e5f6",
    "status": "pending",
    "created_at": "2026-03-20T14:30:22"
  }
}

使用 GET /api/queue/task/{task_id} 輪詢任務狀態。

GET /api/queue/status

取得佇列目前狀態。

API 金鑰管理

端點說明
GET /api/v1/keys列出所有金鑰
POST /api/v1/keys建立新金鑰
DELETE /api/v1/keys/{id}刪除金鑰
POST /api/v1/keys/{id}/revoke撤銷金鑰

專案管理

端點說明
GET /api/projects列出所有專案
POST /api/projects建立新專案
GET /api/projects/{id}取得專案詳情
PUT /api/projects/{id}更新專案
DELETE /api/projects/{id}刪除專案
POST /api/projects/{id}/images加入圖片
PUT /api/projects/{id}/images/{filename}/rate評分圖片 (1-5)
GET /api/projects/{id}/stats專案統計

作品集管理

端點說明
GET /api/galleries列出所有作品集
POST /api/galleries建立作品集
GET /api/galleries/{id}取得作品集
PUT /api/galleries/{id}更新作品集
POST /api/galleries/{id}/images加入圖片

錯誤碼

HTTP StatusCode說明
401MISSING_API_KEY未提供 API 金鑰
401INVALID_API_KEY無效的 API 金鑰
403INSUFFICIENT_PERMISSIONS金鑰權限不足
503-模型尚未載入

速率限制

每個 API 金鑰預設限制為每分鐘 60 次請求。超過限制將回傳 429 Too Many Requests。

可在金鑰建立時自訂限制額度。
